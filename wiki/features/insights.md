# Insights & Analytics

The Insights page transforms your transaction data into actionable knowledge about your spending habits.

---

## Overview

Insights help you answer questions like:
- Where does my money go?
- Am I spending more than usual?
- Which categories need attention?
- How has my spending changed over time?

---

## Available Insights

### 1. Monthly Spending Breakdown

**What it shows:** Pie chart of spending by category for the current month.

**How to read it:**
- Larger slices = more spending
- Colors match category colors
- Hover for exact amounts and percentages

**Insights to look for:**
- Is one category dominating?
- Any unexpected large slices?
- Categories that should be smaller?

---

### 2. Spending Trends

**What it shows:** Line or bar chart of total spending over time.

**Time periods:**
- Last 7 days
- Last 30 days
- Last 6 months
- Last 12 months

**Insights to look for:**
- Upward trend = spending increasing
- Downward trend = spending decreasing
- Spikes = unusual months (holidays, big purchases)

---

### 3. Category Comparison

**What it shows:** Compare spending across categories over time.

**Use this to:**
- See if a category is growing
- Identify seasonal patterns
- Compare before/after lifestyle changes

---

### 4. Top Expenses

**What it shows:** Your largest individual transactions.

**Why it matters:**
- Big expenses have outsized impact
- Helps identify one-time vs recurring big expenses
- Spot potential areas for savings

---

### 5. Budget Progress

**What it shows:** How much of each budget has been used.

**Visual indicators:**
- Green: On track (<75% used)
- Yellow: Caution (75-90% used)
- Red: Over budget (>90% or exceeded)

---

## Reading the Charts

### Pie Charts

Best for: Category distribution
- Shows proportions
- Quick visual of where money goes
- Hover for details

### Bar Charts

Best for: Comparisons
- Compare months side by side
- Compare categories
- Easy to see differences

### Line Charts

Best for: Trends over time
- Shows direction (up/down)
- Identifies patterns
- Spots anomalies

## Using Insights Effectively

### Weekly Check

Quick 2-minute review:
1. Glance at category breakdown
2. Note any surprises
3. Check if on track for budgets

### Monthly Analysis

Deeper 10-minute review:
1. Compare to last month
2. Review each category's spending
3. Identify areas for improvement
4. Adjust budgets if needed

### Quarterly Review

Strategic 30-minute session:
1. Review 3-month trends
2. Analyze category patterns
3. Evaluate financial goals
4. Plan for next quarter

## Common Patterns to Watch

### Lifestyle Creep

**Pattern:** Gradual increase in spending over months
**Cause:** Income increases → spending increases
**Action:** Set firm budgets, review wants vs needs

### Category Drift

**Pattern:** One category growing unexpectedly
**Cause:** Changed habits, new subscriptions
**Action:** Review transactions, set category budget

### Spike Pattern

**Pattern:** Occasional large spikes in spending
**Cause:** Big purchases, emergencies, holidays
**Action:** Plan for known spikes, build emergency fund

### Seasonal Patterns

**Pattern:** Same months always higher
**Cause:** Holidays, seasons, annual expenses
**Action:** Budget higher for those months

## Actionable Insights

### Finding Savings

1. **Identify top 3 categories** - Focus efforts here
2. **Review each category's transactions**
3. **Ask:** Can I reduce this by 10%?
4. **Set a budget** to enforce the reduction

### Tracking Progress

1. **Set a goal** (e.g., reduce dining out by ₹2,000)
2. **Create a budget** for that category
3. **Check weekly** - are you on track?
4. **Review monthly** - celebrate wins, adjust if needed

### Spotting Problems

Look for:
- Categories with no budget that are growing
- Spending patterns you don't recognize
- Transactions you don't remember

## Insights Best Practices

### 1. Accurate Data = Accurate Insights

- Log all transactions
- Use correct categories
- Include all accounts

### 2. Regular Review Schedule

- Daily: Log transactions
- Weekly: Quick insights check
- Monthly: Deep analysis

### 3. Take Action

Insights are only valuable if you act on them:
- Set budgets based on what you learn
- Make concrete changes
- Track if changes are working

### 4. Don't Obsess

- Financial health is a marathon, not a sprint
- Small improvements compound
- Focus on trends, not day-to-day
