# First Steps with Kharcha

Congratulations on setting up Kharcha! This guide will walk you through your first actions to start tracking your finances effectively.

---

## Overview

After logging in, you'll want to:

1. **Set up your accounts** (where your money lives)
2. **Create expense categories** (how you categorize spending)
3. **Add your first transaction**
4. **Explore the dashboard**

Let's go through each step.

---

## 1. Creating Accounts

Accounts represent where your money is stored or comes from. Kharcha supports various account types:

| Account Type | Example Use |
|--------------|-------------|
| **Cash** | Physical wallet, petty cash |
| **Bank** | Savings account, current account |
| **Credit Card** | Visa, Mastercard, Amex |
| **UPI** | Google Pay, PhonePe, Paytm |
| **Wallet** | Digital wallets like Amazon Pay |
| **Loan** | Track loan payments |
| **Other** | Any other financial account |

### How to Create an Account

1. Navigate to **Accounts** from the sidebar or bottom navigation
2. Click the **"Add Account"** button
3. Fill in the details:
   - **Name**: Give it a recognizable name (e.g., "HDFC Savings", "Personal Wallet")
   - **Type**: Select the account type
   - **Color**: Choose a color for easy identification
4. Click **Save**

### Best Practices

- ‚úÖ Create separate accounts for each bank account you have
- ‚úÖ Create a "Cash" account for physical money
- ‚úÖ Use meaningful names you'll recognize
- ‚úÖ Assign different colors for quick visual distinction

## 3. Adding Your First Transaction

Now let's record an expense!

### Quick Add (from Dashboard)

1. On the **Dashboard**, click the **"Add Transaction"** button
2. Fill in the transaction details:
   - **Amount**: How much did you spend?
   - **Account**: Which account was used?
   - **Category**: What type of expense is this?
   - **Date**: When did this happen? (defaults to today)
   - **Note**: Optional description (e.g., "Lunch with John")
3. Click **Save**

### From Transactions Page

1. Navigate to **Transactions**
2. Click **"Add Transaction"**
3. Complete the form as above

### Transaction Fields Explained

| Field | Required | Description |
|-------|----------|-------------|
| Amount | ‚úÖ | The transaction amount |
| Account | ‚úÖ | Which account to deduct from |
| Category | ‚úÖ | Type of expense |
| Date | ‚úÖ | Transaction date and time |
| Note | ‚ùå | Optional description |

## 4. Adding Subscriptions

For recurring expenses like Netflix, Spotify, or gym memberships:

1. From the **Dashboard**, click **"Add Subscription"**
2. Fill in:
   - **Name**: Subscription name
   - **Amount**: Monthly/yearly cost
   - **Category**: Type of expense
   - **Billing Date**: When it renews
3. Click **Save**

Subscriptions help you:
- Track recurring expenses
- Remember when payments are due
- See total subscription costs

## 5. Exploring the Dashboard

The Dashboard gives you an at-a-glance view of your finances:

### Dashboard Widgets

| Widget | What It Shows |
|--------|---------------|
| **Total Balance** | Sum of all account balances |
| **This Month's Spending** | Total spent this month |
| **Category Breakdown** | Spending by category |
| **Recent Transactions** | Latest transactions |
| **Top Spending Categories** | Where most money goes |

### Reading the Charts

- **Pie Chart**: Shows spending distribution across categories
- **Bar Chart**: Compares spending across time periods
- **Trend Line**: Shows spending patterns over time

---

## 6. Setting Up Budgets

Budgets help you control spending in specific categories.

### Creating a Budget

1. Go to **Insights** or **Budgets**
2. Click **"Set Budget"**
3. Configure:
   - **Category**: Which category to budget
   - **Amount**: Monthly budget limit
   - **Month**: Which month this applies to
4. Click **Save**

### Budget Tracking

- See progress bars showing how much of your budget is used
- Get warnings when approaching budget limits
- Review which categories consistently exceed budgets
