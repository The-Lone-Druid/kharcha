**Scenario:** You want to be reminded before subscriptions auto-renew so you can cancel if needed.

### Setup

   - Settings â†’ Notifications â†’ Enable all notifications


- **Subscription Renewals** - Control renewal reminders

## Monthly Financial Review

**Scenario:** You want to do a thorough monthly review of your finances.

### Monthly Review Checklist

**Week 1 of Each Month:**

- [ ] Review last month's total spending
- [ ] Compare to previous months
- [ ] Check budget adherence per category
- [ ] Identify overspending areas
- [ ] Update budgets if needed
- [ ] Review subscriptions (still needed?)
- [ ] Set goals for this month

### Using Kharcha for Review

1. **Dashboard** - Get the overview
2. **Insights** - Analyze trends
3. **Transactions** - Filter by last month
4. **Budgets** - Check progress/overages

### Questions to Ask

- What was my biggest expense category?
- Any unexpected large expenses?
- Did I stick to my budgets?
- What can I reduce next month?
